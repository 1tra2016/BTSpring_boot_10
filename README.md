# 5 URL chuẩn RESTful cho các thao tác sau:
- Lấy danh sách toàn bộ bác sĩ: 
GET /api/v1/doctors

- Lấy thông tin chi tiết một bác sĩ theo ID: 
GET /api/v1/doctors/{id}

- Thêm mới một bác sĩ:
POST /api/v1/doctors

- Cập nhật thông tin bác sĩ:
POST /api/v1/doctors

- Xóa bác sĩ khỏi hệ thống:
DELETE /api/v1/doctors/{id}
